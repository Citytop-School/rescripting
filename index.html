  • Page Titles : Titles in html.
  • Paragraphs : Structuring your contents.
  • Headings : The six levels of headings.
  • Lists : How to define ordered and unordered lists.
  • Links : How to link html pages to navigate from one to another.
  • Images : Adding something a bit more than text.
  • Table : Tabulating on the web.
